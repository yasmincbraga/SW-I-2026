Sobre

O PHP (Hypertext Preprocessor) é uma linguagem de script open-source de uso geral, amplamente utilizada no desenvolvimento web back-end

Executada no servidor, ela permite criar sites dinâmicos, processar formulários e conectar a bancos de dados, sendo fácil de aprender e com custo de hospedagem acessível.
